API Threat Detection System – Project Demo

Student: Anand

Description:
This project demonstrates an AI-based API threat detection system that analyzes behavioral patterns of API traffic to identify suspicious activity.

Files Included:
- API_Threat_Detection_Presentation.pptx
- api_server.py
- train_hybrid_model.py
- dashboard.html

Steps to Run the Demo:

1. Train the model
python train_hybrid_model.py

2. Start the monitoring system
python api_server.py

The dashboard will open automatically in the browser.
